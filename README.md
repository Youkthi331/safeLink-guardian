# SafeLink Guardian 🔐

SafeLink Guardian is a browser extension designed to protect users from phishing attacks by detecting suspicious URLs and warning users before they interact with potentially dangerous websites.

---

## 🚨 Problem

Phishing attacks are one of the most common cyber threats today.  
Attackers create fake websites that look like legitimate banking, login, or service pages to steal sensitive information such as:

- Passwords
- Banking details
- Personal data

Many users, especially elderly or non-technical users, cannot easily identify malicious links.

---

## 💡 Solution

SafeLink Guardian automatically detects suspicious keywords in URLs and warns the user immediately with:

- 🚫 A red warning screen
- 🔊 Voice alerts in multiple languages
- ⚠️ Real-time phishing detection

This helps prevent users from entering sensitive information on fake websites.

---

## ⚙️ Features

- Real-time phishing link detection
- Automatic redirect to warning page
- Multilingual voice alerts
- Browser extension integration
- Lightweight and fast detection system
